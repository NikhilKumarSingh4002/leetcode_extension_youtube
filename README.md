LeetCode YouTube Solutions (user-key)
====================================

This Chrome extension shows YouTube solution videos inside LeetCode problem pages.
It uses a user-provided YouTube Data API key (stored locally in chrome.storage.sync).

How to use:
1. Install the extension.
2. Open Options and paste your YouTube Data API key (see instructions in Options page).
3. Visit a LeetCode problem page (https://leetcode.com/problems/<slug>/) and the video carousel will appear near the top.

Privacy:
- The extension stores the user's API key only in Chrome storage (on their machine/account). It is not sent to any external server by this extension.
