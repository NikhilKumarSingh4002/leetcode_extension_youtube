Privacy Policy – LeetCode YouTube Solutions (User Key Edition)

This Chrome extension helps users view YouTube solution videos directly on LeetCode problem pages.

Information We Collect

We do not collect any personal data.

  • The YouTube API key you provide is stored locally in Chrome sync storage on your device.
  • The key is used only to make requests directly to the official YouTube Data API.
  • No analytics, tracking, or third-party storage is performed.

Data Sharing

No information is shared with anyone. All API requests go directly from your browser to YouTube’s API servers.

Removing Your Data

You may remove your API key at any time from the extension’s Options page or by uninstalling the extension.
